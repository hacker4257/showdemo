# PharmaTrade 项目说明

## 项目概述

PharmaTrade 是一个专业的医药外贸电商平台，参考 ChemExpress 网站设计，用于展示和销售医药化合物、生物制品等研究用化学品。

## 技术特点

### 前端技术
- **Next.js 14+** - 使用最新的 App Router 架构
- **React 18+** - 现代化的 React 开发
- **TypeScript** - 类型安全
- **Tailwind CSS** - 实用优先的 CSS 框架
- **Lucide React** - 现代化图标库

### 后端技术
- **Prisma ORM** - 类型安全的数据库访问
- **PostgreSQL** - 可靠的关系型数据库
- **Next.js API Routes** - 服务端 API
- **Zod** - 运行时类型验证

### 设计特点
- 响应式设计，支持移动端和桌面端
- 专业的医药行业配色（蓝白色系）
- 清晰的信息架构
- 用户友好的交互设计

## 核心功能

### 1. 首页
- Hero 区域展示公司定位
- 核心优势展示（高纯度、质量保证、全球配送、竞争力价格）
- 产品分类快速导航
- CTA（行动号召）区域

### 2. 产品目录
- 搜索功能（产品名称、CAS号）
- 分类筛选
- 纯度筛选
- 库存状态筛选
- 产品卡片展示（图片、名称、CAS号、纯度）
- 分页功能

### 3. 产品详情
- 完整产品信息展示
- 化学结构图占位符
- 基本信息（CAS号、分子式、分子量、纯度）
- 产品描述
- 规格参数
- 应用领域
- 询价功能（模态框）
- 侧边栏联系信息

### 4. 询价系统
- 模态框表单
- 必填字段验证
- 客户信息收集（姓名、邮箱、公司、电话、国家）
- 数量和留言
- 提交成功反馈
- API 后端处理

### 5. 联系页面
- 联系信息展示（邮箱、电话、地址、营业时间）
- 联系表单
- 表单验证
- 提交成功反馈

### 6. 关于我们
- 公司介绍
- 统计数据展示
- 核心价值观
- 认证和合规信息

## 数据库设计

### 核心表结构

**Product（产品）**
- 中英文名称
- CAS号（唯一索引）
- 分子式、分子量
- 纯度
- 中英文描述
- 规格参数（JSON）
- 应用领域
- 图片URL
- 分类关联
- 库存状态
- 精选标记

**Category（分类）**
- 中英文名称
- URL slug
- 中英文描述

**Inquiry（询价）**
- 产品关联
- 客户信息（姓名、邮箱、公司、电话、国家）
- 询价详情（数量、留言）
- 状态跟踪（待处理、已联系、已报价、已完成、已取消）
- 管理员备注

**Contact（联系表单）**
- 客户信息
- 主题和消息
- 状态跟踪

**User（用户）**
- 管理员账号
- 角色权限

## API 端点

### 询价 API
- `POST /api/inquiries` - 提交询价
- `GET /api/inquiries` - 获取询价列表（需认证）

### 联系 API
- `POST /api/contact` - 提交联系表单

## 部署建议

### 推荐方案：Vercel
- 零配置部署
- 自动 HTTPS
- 全球 CDN
- 环境变量管理
- 预览部署

### 数据库托管
- Vercel Postgres
- Supabase
- Railway
- 自托管 PostgreSQL

### 环境变量配置
部署时需要配置：
- `DATABASE_URL` - 数据库连接
- `NEXTAUTH_URL` - 网站URL
- `NEXTAUTH_SECRET` - 认证密钥
- SMTP 配置（邮件通知）
