# 多语言功能说明

## ✅ 已实现

PharmaTrade 现已支持中英文双语切换！

### 功能特点

1. **语言切换器**
   - 位置：页面顶部右上角
   - 支持语言：English（英文）、中文
   - 实时切换，无需刷新页面

2. **持久化存储**
   - 语言选择自动保存到浏览器 localStorage
   - 下次访问自动恢复上次选择的语言

3. **全站翻译**
   - ✅ 导航栏
   - ✅ 首页（Hero、特性、分类、CTA）
   - 📝 产品页面（待更新）
   - 📝 联系页面（待更新）
   - 📝 关于页面（待更新）

### 使用方法

1. 访问网站：http://localhost:3001
2. 点击右上角的语言选择器
3. 选择 "中文" 或 "English"
4. 页面内容立即切换

### 技术实现

- **Context API** - React Context 管理全局语言状态
- **自定义 Hook** - `useI18n()` 提供语言切换和翻译功能
- **JSON 翻译文件** - 结构化的翻译内容
  - `messages/en.json` - 英文翻译
  - `messages/zh.json` - 中文翻译

### 文件结构

```
pharma-trade/
├── lib/
│   └── i18n/
│       └── context.tsx          # I18n Context Provider
├── messages/
│   ├── en.json                  # 英文翻译
│   └── zh.json                  # 中文翻译
├── components/
│   └── layout/
│       └── Header.tsx           # 带语言切换的导航栏
└── app/
    ├── layout.tsx               # 包含 I18nProvider
    └── page.tsx                 # 多语言首页
```

### 添加新翻译

1. 在 `messages/en.json` 和 `messages/zh.json` 中添加新的翻译键值对
2. 在组件中使用 `useI18n()` Hook：

```tsx
'use client'

import { useI18n } from '@/lib/i18n/context'

export default function MyComponent() {
  const { t } = useI18n()

  return (
    <div>
      <h1>{t.mySection.title}</h1>
      <p>{t.mySection.description}</p>
    </div>
  )
}
```

### 扩展语言

要添加新语言（如日语、韩语）：

1. 创建新的翻译文件：`messages/ja.json`
2. 更新 `lib/i18n/context.tsx`：
   ```tsx
   import ja from '@/messages/ja.json'
   const messages = { en, zh, ja }
   ```
3. 更新语言选择器添加新选项
